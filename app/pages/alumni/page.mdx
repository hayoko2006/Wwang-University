### GitHub

如果您想在您的 GitHub 个人资料页面上展示神圣大旺帝国旺学，请使用我们 GitHub 仓库中的 **"校友申请"** 模板来请求加入该组织。

<Button variant="default">
  [GitHub](https://github.com/WildChickenUniversity/WildChickenUniversity/issues/new?template=apply.yaml)
</Button>

如果您的申请被接受，您将收到一封带有校友团队邀请链接的录取邮件。
或者，您可以尝试 [此链接](https://github.com/orgs/WildChickenUniversity/invitation?via_email=1)。
如果两者都对您不起作用，请直接联系我们。

如果您不小心创建了一个空问题，请关闭它并使用提供的模板打开一个新的工单。

### 校友邮箱

运行电子邮件服务很容易，但防止滥用很难。我们无法阻止人们使用旺学邮箱申请真正的教育福利。

所以，不，我们没有任何教育邮箱。如果您想通过旺学获得教育特权，请自行寻找其他途径。
